Assignments

Checkpoints (50 points)

  1. 26 Original Photos (5 points):
    Have 26 original photos ready to prepare in class for the Alphabet assignment. They should be photos you took yourself and be files in the original quality that they were captured in (do not edit them yet).

  2. Code Academy Units 1-3 (5 points):
    Complete the Code Academy web tutorials units 1-3. This should familiarize you with basic syntax of HTML markup and composing a simple web page.

  3. Code Academy Units 4-6 (5 points):
    Complete the Code Academy web tutorials units 4-6. This should familiarize you with basic syntax of CSS declarations and how to do simple styling of web elements.

  4. Generate and Implement a Custom Web Font (5 points):
    Use the FontSquirrel webfont generator to create the proper file formats and CSS for a webfont using a TTF/OTF font file. Test the file to make sure it can be applied to your home page.

  5. 20 Original Images (5 points):
    Create 20 original images along a common theme to use in the Gallery assignment. They can be existing images of your own creation or new ones made for this assignment. They should be at least 2000 x 2000 in dimension so that the large version will display nice and sharp on large, high resolution displays.

  6. Narrative Subject and Storyboard (5 points):
    Come to class with at least one idea for a subject to use as the basis for your scrolling site, as well as a basic sketch of some different states of the narrative of your scrolling site.

  7. Data Model Tables (5 points):
    Have a plan prepared about how you will organize your data in a table format for your design blog, identifying the fields used for each entry as well as their corresponding data variable type.

  8. Moodboard, Wireframe (5 points):
    Come to class with a moodboard of imagery that inspires the visual direction of your portfolio site. This can be still images or links to web sites you like (or screen caps with links would be even better). Create some posts on your tumblr with your moodboard items. In addition to the moodboard, begin some rudimentary wireframing of how you imagine your portfolio site content being structurally organized.

  9. Scaffold Pages and Data Model Tables (5 points):
    Have some basic code in place identifying what PHP/CSS/HTML files you will need to compose your site, as well as how they will be connected together. Give your database the same treatment by figuring out how you will organize your information in table format.

  10. Stylesheet Refinement (5 points):
    You should have your site functionally working and content being pulled from your database and rendered dynamically in the browser. At this point you should have a plan on how to refine your CSS to accomplish the visual design you desire.

Exercises (40 points)

  1. Top Five (10 points):
    Collect links to five web sites that you find best convey the following qualities: (1) visual design, (2) motion graphics, (3) mobile design, (4) typography, (5) needs updating. a link to your tumblr when it has been posted.

  2. Alphabet (10 points):
    Take 26 original photos of typography to generate an alphabet from A to Z. Prepare your images in Photoshop and optimize their resolution/compression for your blog. Prepare your tumblr account to automatically post the photos to your blog one an hour in sequential order. Have the posts start 26 hours before the class session the assignment is due.

  3. Just the DIVs (10 points):
    Make a page using at least 20 different divs, using absolute, relative, static, and fixed positioning. Play around with color, positioning and dimensions using pixels and percentages, as well as vw/vh values.

  4. Style an Article (10 points):
    Copy an article that has body text, title, subtitle, publishing information (author, date/time, tags, etc), and at least one photo, and restyle the article using your own CSS.

Projects (110 points)

  1. Your Homepage (20 points):
    Make a responsive website that will be the landing page for your future assignments. You must include a visual identity for yourself (colors, fonts, moodboard, aesthetic), an avatar/logo/photo for the front page to link to your home page, an about yourself paragraph with some representative image, a section for your past and future assignments, and navigation to your tumblr blog.

  2. Gallery (20 points):
    Make a gallery of images along a common theme. Your gallery should have some kind of index (page showing smaller versions of images in order to navigate them) as well as a way of viewing each image in itâ€™s largest, fullest state.

  3. Scrolling Site (20 points):
    Make a narrative site with at least 5 full viewport sections that utilizes scrolling as a navigation method and parallax effects for visual elements. Integrate CSS3 effects, transitions, animations, and jQuery effects where appropriate.

  4. Design Blog (20 points):
    Create a blog about a designer or design movement. The style of the blog must represent the designer through graphics and text. In the end it must have at least 10 sequential blog entries. The blog must use PHP to render HTML output and use content stored in an SQL table.

  5. Portfolio (30 points):
    Create a portfolio website for yourself or an artist/designer you admire. The portfolio must contain at least 10 different entries, each of which having an image, title, date, and description as well as any other data qualities you feel it should include (tags, links, etc). The site must have a portfolio index, portfolio item template page, and a separate about page. Content for the site should be stored in SQL tables.
