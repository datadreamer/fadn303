These are some divs with different position values.

The top four divs (red-yellow) are using the CSS position property to define their behavior and arrangement.

The first div (red) is using the default position:static. The second is using position:relative to shift it to the left. The third is using position:absolute to tack the div to the page. The fourth (yellow) is using position:fixed to tack the div to the browser window.

The next four divs (blue) are using float:left; to determine their arrangement.

The bottom four divs (green) are using float:left; and have a fixed width/height, causing them to wrap with the browser window.

Scroll the page and/or drag the browser window around to see how it affects the div arrangement.
