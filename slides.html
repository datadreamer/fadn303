Slides

  • Week 1: History of the Web (~550mb, 1 hour of video):
    An overview of the development of hypermedia, early networking technologies, consumer introduction to the internet, and the browser wars of the 1990s.
  • Week 2: Servers, Clients and Protocols:
    An introduction to servers, services, clients, protocols, ports, http, and ftp.
  • Week 2: Files, Images and Organization:
    Descriptions of different file types used in the course, as well as discussion of image compression and best practices for organization of web files.
  • Week 3: HTML/CSS Basics:
    The basic syntax of HTML and CSS documents as well as some wisdom regarding organization of web site files.
  • Week 3: Containers & Positioning:
    Outlining block and inline elements, as well as how to arrange them on screen using the position property or the float property.
  • Week 4: CSS Classes & IDs:
    How to declare styles for groups of HTML elements and specific HTML elements.
  • Week 4: Browser Tools:
    What browser developer tools are available and how to make effective use of them in constructing and debugging your website.
  • Week 5: Code Ethics:
    Discussion regarding the ethical use of code, sources for learning about programming, and how to implement coding solutions.
  • Week 5: Page Composition & Wireframing:
    Example of breaking down a site structure into it's elements and figuring out how to effectively organize content.
