3. Progressive Enhancement

Media Queries


Media Queries allow you to use breakpoints to specify different CSS styles for different window dimensions.
Responding to Device Resolution

#mediatest1

@media screen and (min-width: 640px) {
  #mediatest1 { background-color: lightgreen; }
}
@media screen and (min-width: 960px) {
  #mediatest1 { background-color: yellow; }
}
@media screen and (min-width: 1280px) {
  #mediatest1 { background-color: orange; }
}
@media screen and (min-width: 1440px) {
  #mediatest1 { background-color: red; }
}

CSS3 Box Sizing


The CSS3 box-sizing property allows you to include the padding and border in an element's total width and height.

Before Box Sizing


This div is smaller because it has no padding.
This div is larger because its padding is added to its width and height.

#div1 {
  width: 300px;
  height: 100px;
  border: 1px solid blue;
}

.div2 {
  width: 300px;
  height: 100px;
  padding: 50px;
  border: 1px solid red;
}

After Box Sizing


This div is unchanged.
This div is now the same dimensions, as the padding is included in the width and height.

#div1 {
  width: 300px;
  height: 100px;
  border: 1px solid blue;
  box-sizing: border-box;
}

.div2 {
  width: 300px;
  height: 100px;
  padding: 50px;
  border: 1px solid red;
  box-sizing: border-box;
}
