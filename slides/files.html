.html

  • Plain text file.
  • Contains hypertext markup.
  • Used to describe the structure of a web page.

.css

  • Plain text file.
  • Contains cascading stylesheets.
  • Used to visually style elements of a web page.

.js

  • Plain text file.
  • Contains javascript.
  • Used to describe the behavior of elements on a web page.

.php

  • Plain text file.
  • Contains php hypertext preprocessor code.
  • Used to dynamically generate html/css/js code with server content.

.jpg

  • Lossy bitmap image format.
  • Good for photos with a lot of colors and detail.
  • 24-bit color (16.7 million colors)

.png

  • Lossless bitmap image format.
  • Good for images with fewer colors, sharp edges, and transparency.
  • 32-bit color (16.7 million colors + 8-bit alpha channel)

.gif

  • Lossless bitmap image format.
  • Good for images with few colors, basic transparency, and animation.
  • 8-bit color (256 colors + alpha channel)

.mp4

  • Lossy video format.
  • Good for full color animation.
  • 24-bit color (16.7 million colors), any FPS.

.ttf

  • TrueType Font.
  • Developed by Apple and Microsoft in the late 1980s.
  • Standard format for font distribution.

.otf

  • OpenType Font.
  • Developed by Microsoft and Adobe in 1996.
  • Standard format for font distribution.

.eot

  • Embedded OpenType Font.
  • Developed by Microsoft in 2007.
  • Proprietary and supported only by Internet Explorer.

.woff

  • Web Open Font Format.
  • Developed by W3C in 2009.
  • Intended to be the next open font format to be supported by all browsers.

.svg

  • Scalable Vector Graphics.
  • Developed by W3C in 1999.
  • Open vector image format also used to contain characters for font files.
  • Images and their behaviors are defined in XML text files.
