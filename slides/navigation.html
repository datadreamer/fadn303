Navigation Styles


Part of wireframing your site is making a UI/UX flowchart to organize the way the different sections of your web site connect. These might be different html documents linked together to create a site, or a single html document with multiple distinct sections.

Pagination


Pagination is a design technique where you attempt to fit everything onto the screen at once, as if reading a printed page. Pagination requires the use of responsive design to accommodate displays with different dimensions and resolutions.

Demo

vw/vh


In order to appropriately size paginated sections, we need to be able to reliably measure the viewport width and height. vw and vh are css measurement types used to position and size elements based on the complete viewport dimensions.
100vw = 100% viewport width
100vh = 100% viewport height

vw/vh is different from using the % measurement because they are always based on the viewport dimensions, regardless of how nested the html element is that you are applying it to.

Paginated Linking


Linking to different sections within the page is just as easy as linking to other pages. This gives the user the option to scroll through all the content or use a navigation menu to select a specific section to view.
