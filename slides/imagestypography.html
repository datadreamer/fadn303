Image Types

JPEG
(1986)

  • Joint Photographic Experts Group
  • Lossy Compression
  • 24-bit color
  • Good for photos.

PNG
(1996)

  • Portable Network Graphics
  • Lossless Compression
  • 32-bit color (includes 8-bit alpha channel)
  • Good for illustration, text, and transparency.

GIF
(1987)

  • Graphics Interchange Format
  • Lossy Compression
  • 8-bit indexed color
  • Okay for animations, but dying out.

Color Types

Color Name

  • Plain text names
  • 140 available
  • DeepPink
  • DodgerBlue
  • GreenYellow

Hexadecimal

  • Uses alphanumeric code to specify RGB values.
  • 24-bit color
  • #FF1493
  • #1E90FF
  • #ADFF2F

RGBA/HSLA

  • Uses 8-bit numbers for color channels.
  • 32-bit color (can contain alpha channel).
  • rgb(255, 20, 147)
  • rgb(30, 144, 255)
  • rgb(173, 255, 47)
