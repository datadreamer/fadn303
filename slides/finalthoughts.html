What to do next?

Frontend Development


AJAX

Asynchronous Javascript and XML is used for a web page to communicate with a server and update its content without refreshing the browser window.

FRP

Functional Reactive Programming (FRP) is a programming paradigm for reactive programming (asynchronous dataflow programming) using the building blocks of functional programming (e.g. map, reduce, filter).

Frameworks

Javascript Frameworks are Javascript libraries that provide full-stack capabilities, and are specifically optimized for developing particular applications.

HTML5 Canvas


p5.js

P5.js is a pure javascript implementation of the Processing language. It includes many of its drawing functionalities and has additional support for accessing the DOM and HTML5 media elements.

Two.js

Two.js is a two-dimensional drawing api geared towards modern web browsers. It is renderer agnostic enabling the same api to draw in multiple contexts: svg, canvas, and webgl.

Paper.js

Paper.js is an open source vector graphics scripting framework with its own document object model, allowing object oriented control over drawable elements.

HTML5 Media


Video/Audio

Playback video and audio from a variety of formats directly in the browser. Content can be embedded in elements, and thus can be styled and transformed like anything else in the DOM.

getUserMedia

The getUserMedia function of Javascript allows access to all sorts of media capture, from the users camera, microphone, geolocation, even device orientation.

WebRTC

WebRTC (Real-time communication) is used to communicate media across devices without plugins or the need for a central server.

WebGL


Three.js

Three.js is a powerful Javascript library for doing 3D rendering in WebGL. It takes a very traditional approach to OpenGL applications, and has support for importing 3D models and rendering scenes with GLSL shaders.

BabylonJS

Babylon.js is a Javascript framework specifically intended for building 3D games with HTML5, WebGL and web audio. Not only does it render scenes, it fully supports collision detection and has an integrated physics engine.

A-Frame

A-Frame is a framework that allows you to use custom markup to define 3D objects for rendering in a WebGL context for desktop or VR.
