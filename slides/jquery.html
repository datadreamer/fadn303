Javascript

What is Javascript?

  • Industry standard scripting language of the web.
  • Used to program behavior of web pages.
  • High-level, dynamic, untyped, and interpreted.
  • Chunks of code are distributed as javascript libraries.

jQuery

What is jQuery?

  • Javascript library for programming page behavior.
  • Shortcuts for performing Javascript tasks.
  • HTML/DOM/CSS manipulation.
  • Events, listeners, and loops.
  • Effects and Animations.
  • Asynchronous Javascript and XML (AJAX).
  • Thousands of plugins.
Including jQuery


To use a javascript library, it has to be included in your HTML document. This is similar to referencing external CSS files, where it has to be declared inside the <head> tags. This will allow you to make use of all of jQuery's methods.


<head>
  <script src="jquery-1.11.3.min.js"></script>
</head>

or

<head>
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>
</head>
Basic Syntax


Basic syntax uses the $(selector).action() format.

The $ symbol indicates that you're making use of jQuery. The selector is the CSS selector used to declare the element(s) that will be affected. The action is the method used to apply some change to the selected element(s).


$(function(){
  
  $("#jstest1a").click(function(){
    $("#jstest1b").toggle();
  });

});

Events


jQuery can be used to write code that occurs on particular events.


$(function(){

  $("#jstest2a").on({
    mouseenter: function(){
      $("#jstest2b").css("background-color", "red");
    },
    mouseleave: function(){
      $("#jstest2b").css("background-color", "#0066cc");
    },
    click: function(){
      $("#jstest2b").css("background-color", "green");
    }
  });
});

    mouseleave: function(){
       $("#jstest3b").show();
    },
    click: function(){
       $("#jstest3c").toggle();

Animating


Elements can be animated by specifying different css properties to affect.


$(function(){

  $("#jstest5a").on({
    mouseenter: function(){
       $("#jstest6b").animate({
         marginLeft: '35%'
       });
    },
    mouseleave: function(){
       $("#jstest6b").animate({
         marginLeft: '0%'
       });
     },
    click: function(){
       $("#jstest6b").animate({
         marginTop: '+=20px'
       });
     }
  });

});

Multiple Animations


Multiple animations can be stacked together.


$(function(){

  $("#jstest7a").on({
    mouseenter: function(){
      $("#jstest7b").animate({width:'65%'});
      $("#jstest7b").animate({height:'200px'});
    },
    mouseleave: function(){
      $("#jstest7b").animate({height:'auto'});
      $("#jstest7b").animate({width:'30%'});
     }
  });

});

jQuery Dimensions


jQuery can be used to grab the dimensions of any elements current size.


$(function(){

  $("#jstest11a").click(function(){
    var w = $("#jstest11a").width();
    var h = $("#jstest11a").height();
    $("#jstest11a").after("<br/>" + w +" "+ h);
  });

});
