CSS3 Visual Styles

Rounded Corners


Simply round out the edges of your element using the border-radius property.


div {
  border-radius: 25px;
  background: #0099ff;
  padding: 20px;
  width: 200px;
  height: 150px;
}
Rounded!
Rounded Corners

.rctest {

  float:left;
}

Border Images


Create an image that holds tiles for use in borders.


#borderimg {
  border: 10px solid transparent;
  -webkit-border-image: url(border.png) 30 round;
  -o-border-image: url(border.png) 30 round;
  border-image: url(border.png) 30 round;
  padding: 30px;
}

Festive!
Border Images


Stretch the sides for a different look.


#borderimg {
  border: 10px solid transparent;
  -webkit-border-image: url(border.png) 30 stretch;
  -o-border-image: url(border.png) 30 stretch;
  border-image: url(border.png) 30 stretch;
  padding: 30px;
}

Gradients


Apply smooth gradients to the background of your elements.


#grad1 {
  background: -webkit-linear-gradient(#000066, #0099ff);
  background: -o-linear-gradient(#000066, #0099ff);
  background: -moz-linear-gradient(#000066, #0099ff);
  background: linear-gradient(#000066, #0099ff);
  padding: 100px;
}
Vertical
Gradients


Gradiants can be horizontal as well.


#grad1 {
  background: -webkit-linear-gradient(left, #000066, #0099ff);
  background: -o-linear-gradient(right, #000066, #0099ff);
  background: -moz-linear-gradient(right, #000066, #0099ff);
  background: linear-gradient(to right, #000066, #0099ff);
  padding: 100px;
}
Horizontal
Gradients


In fact, gradients can be set to whatever angle your specify.


#grad1 {
  background: -webkit-linear-gradient(45deg, #000066, #0099ff);
  background: -o-linear-gradient(45deg, #000066, #0099ff);
  background: -moz-linear-gradient(45deg, #000066, #0099ff);
  background: linear-gradient(45deg, #000066, #0099ff);
  padding: 100px;
}
Angled
Gradients


Go nuts with colors.


#grad1 {
  background: -webkit-linear-gradient(left, red, orange, yellow, green, blue, indigo, violet);
  background: -o-linear-gradient(right, red, orange, yellow, green, blue, indigo, violet);
  background: -moz-linear-gradient(right, red, orange, yellow, green, blue, indigo, violet);
  background: linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet);
  padding: 100px;
}
Rainbow!
Gradients


Gradiants can be applied radially as well.


#grad1 {
  background: -webkit-radial-gradient(#000066, #0099ff);
  background: -o-radial-gradient(#000066, #0099ff);
  background: -moz-radial-gradient(#000066, #0099ff);
  background: radial-gradient(#000066, #0099ff);
  padding: 100px;
}

Shadows
Text-Shadow


Add a basic drop shadow to your text.


#shadowbox {
  background-color: white;
  color: #0099ff;
  text-shadow: 2px 2px 4px black;
  padding: 30px;
}
Shadow
Text-Shadow


Turn that shadow into a glow.


#shadowbox {
  background-color: transparent;
  color: #0099ff;
  text-shadow: 0px 0px 10px white;
  padding: 30px;
}
Shadow
Text-Shadow


Combine multiple shadows in one line to stack effects.


#shadowbox {
  background-color: transparent;
  color: #0099ff;
  text-shadow: 0 0 10px white, 0 0 25px #00ff99, 0 0 50px #006699;
  padding: 30px;
}
NEON!
Box-Shadow


You can add shadows to the boxes around your elements as well.


#shadowbox {
  background-color: white;
  color: #0099ff;
  box-shadow: 5px 5px 10px black;
  padding: 30px;
}
NEON!

CSS3 Transforms

translate()


This method moves an element from its current position.


div {
  background: rgba(0,180,255,0.5);
  -ms-transform: translate(50px,100px);
  -webkit-transform: translate(50px,100px);
  transform: translate(50px,100px);
}
I've been moved. :D
rotate()


This method rotates an element according to a given degree.


div {
  background: rgba(0,180,255,0.5);
  -ms-transform: rotate(20deg);
  -webkit-transform: rotate(20deg);
  transform: rotate(20deg);
}
I feel sick. :(
scale()


This method increases or decreases the size of an element by a factor.


div {
  background: rgba(0,180,255,0.5);
  -ms-transform: scale(3,5);
  -webkit-transform: scale(3,5);
  transform: scale(3,5);
}
omg big :X
skew(), skewX(), skewY(), matrix()


Different methods for distorting the element.


div {
  background: rgba(0,180,255,0.5);
  -ms-transform: skew(20deg, -30deg);
  -webkit-transform: skew(20deg, -30deg);
  transform: skew(20deg, -30deg);
}
feeling skewy ;)
translateX(), translateY(), translateZ(), translate3d()


Methods for positioning elements in 3D space.


div {
  background: rgba(0,180,255,0.5);
  -webkit-transform: translate3d(-100px, -100px, 200px);
  transform: translate3d(-100px, -100px, 200px);
}
3D positioning
rotateX(), rotateY(), rotateZ(), rotate3d()


Methods for orienting elements in 3D space.


div {
  background: rgba(0,180,255,0.5);
  -webkit-transform: rotate3d(1, 1, 1, 30deg);
  transform: rotate3d(1, 1, 1, 30deg);
}
3D rotation
scaleX(), scaleY(), scaleZ(), scale3d()


Methods for scaling elements in 3D space.


div {
  background: rgba(0,180,255,0.5);
  -webkit-transform: scale3d(2, 2, 2);
  transform: scale3d(2, 2, 2);
}

CSS3 Transitions

Transition


A basic transition just responds when a property is changed.


div {
  width: 100px;
  height: 100px;
  background: #0099ff;
  -webkit-transition: width 2s;
  transition: width 2s;
}

div:hover {
  width: 300px;
}
Multiple Transitions


Declare multiple properties to use transitions between states.


div {
  width: 100px;
  height: 100px;
  background: #0099ff;
  -webkit-transition: width 2s, background 4s;
  transition: width 2s, background 4s;
}

div:hover {
  width: 300px;
  background: red;
}
Transition Delay


Add a delay before the transition kicks in.


div {
  width: 100px;
  height: 100px;
  background: #0099ff;
  -webkit-transition: width 1s, background 1s;
  transition: width 1s, background 1s;
  -webkit-transition-delay: 1s;
  transition-delay: 1s;
}

div:hover {
  width: 300px;
  background: red;
}
Transition + Transformation


Transition between property states and CSS3 transforms.


div {
  width: 100px;
  height: 100px;
  background: #0099ff;
  -webkit-transition: width 2s, height 2s, background 2s, -webkit-transform 2s;
  transition: width 2s, height 2s, background 2s, transform 2s;
}

div:hover {
  width: 300px;
  height: 300px;
  background: red;
  -webkit-transform: rotate(180deg);
  transform: rotate(180deg);
}

CSS3 Animations

Basic @keyframes


Add a delay before the transition kicks in.


@keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}

div {
  width: 200px;
  height: 200px;
  background: red;
  animation-name: example;
  animation-duration: 4s;
}
Multiple @keyframes


Customize the timing of your animation.


@keyframes example2 {
  0% {background-color: red;}
  25% {background-color: yellow;}
  50% {background-color: green;}
  100% {background-color: blue;}
}

div {
  width: 200px;
  height: 200px;
  background: red;
  animation-name: example2;
  animation-duration: 4s;
}
Multiple Properties


Customize the timing of your animation.


@keyframes example3 {
  0% {background-color: red; left:0px; top:0px;}
  25% {background-color: yellow; left:200px; top:0px;}
  50% {background-color: blue; left:200px; top:200px;}
  75% {background-color: green; left:0px; top:200px;}
  100% {background-color: red; left:0px; top:0px;}
}

div {
  width: 200px;
  height: 200px;
  position: relative;
  background: red;
  animation-name: example2;
  animation-duration: 4s;
}
Repeat


Set the animation to repeat


@keyframes example3 {
  0% {background-color: red; left:0px; top:0px;}
  25% {background-color: yellow; left:200px; top:0px;}
  50% {background-color: blue; left:200px; top:200px;}
  75% {background-color: green; left:0px; top:200px;}
  100% {background-color: red; left:0px; top:0px;}
}

div {
  width: 200px;
  height: 200px;
  position: relative;
  background: red;
  animation-name: example2;
  animation-duration: 4s;
  animation-iteration-count: infinite;
}
Reverse


Reverse the direction with one line.


@keyframes example3 {
  0% {background-color: red; left:0px; top:0px;}
  25% {background-color: yellow; left:200px; top:0px;}
  50% {background-color: blue; left:200px; top:200px;}
  75% {background-color: green; left:0px; top:200px;}
  100% {background-color: red; left:0px; top:0px;}
}

div {
  width: 200px;
  height: 200px;
  position: relative;
  background: red;
  animation-name: example2;
  animation-duration: 4s;
  animation-iteration-count: infinite;
  animation-direction: reverse;
}
Alternate


Alternate makes the animation bounce back and forth.


@keyframes example3 {
  0% {background-color: red; left:0px; top:0px;}
  25% {background-color: yellow; left:200px; top:0px;}
  50% {background-color: blue; left:200px; top:200px;}
  75% {background-color: green; left:0px; top:200px;}
  100% {background-color: red; left:0px; top:0px;}
}

div {
  width: 200px;
  height: 200px;
  position: relative;
  background: red;
  animation-name: example2;
  animation-duration: 4s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}
