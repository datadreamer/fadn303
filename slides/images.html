IMG Tag


The <img> tag is the original way to display an image as an html element.
<img src="images/corgismile.jpg" alt="corgismile">
<img src="images/corgismile.jpg" alt="corgismile" id="corgione">

<img src="images/corgismile.jpg" alt="corgismile" id="corgitwo">



#corgione{
  width: 200px;
}
#corgitwo{
  width: 400px;
}

Background-image


The background-image CSS property is a way to fill the background of an html element with an image.


<div id="corgidiv"></div>
#corgidiv {
  background-image: url('images/corgismile.jpg');
  width: 100%;
  padding-top: 50%;
}


<div id="corgidiv"></div>
#corgidiv {
  background-image: url('images/corgismile.jpg');
  background-size: cover;
  width: 100%;
  padding-top: 50%;
}


<div id="corgidiv"></div>
#corgidiv {
  background-image: url('images/corgismile.jpg');
  background-size: contain;
  width: 100%;
  padding-top: 50%;
}

}


<div id="corgidiv"></div>
