Font-Family


The font-family css declaration is used to specify the font face to use for a particular element. You can specify multiple fonts in order to provide fallbacks in case fonts are not found or are not able to load.

This is my serif paragraph.




p {
  font-family: "Times New Roman", serif;
}

This is my sans-serif paragraph.




p {
  font-family: Helvetica, Arial, sans;
}

Font-Size


The font-size css declaration is used to specify the font size to use for a particular element.

This is my small text.




p {
  font-size: 1.5em;
}

This is my large text.




p {
  font-size: 4em;
}

Font-Weight


The font-weight css declaration is used to specify the weight of the font to be used. This works best when you have a font that comes in a pack with multiple weights.

This is my light text.




p {
  font-weight: 200;
}

This is my heavy text.




p {
  font-weight: 700;
}

Font-Style


The font-style css declaration is used to specify whether a font should be italic or bold.

This is my bold text.




p {
  font-style: bold;
}

This is my italic text.




p {
  font-style: italic;
}

Letter-Spacing


The letter-spacing css declaration is used to adjust tracking for paragraphs.

Spaced Out.




p {
  letter-spacing: 1em;
}

Pushed Together.




p {
  letter-spacing: -0.1em;
}

Embedding Fonts


By default, web pages will try to use fonts that are installed on your local system. In order to use custom fonts so that everyone can see them, they must be embedded in your CSS as web fonts.

My Custom Web Font.



@font-face {
  font-family: 'dinm';
  src: url('fonts/dinm.eot');
  src: url('fonts/dinm.eot#iefix') format('embedded-opentype'),
    url('fonts/dinm.woff') format('woff'),
    url('fonts/dinm.ttf') format('truetype'),
    url('fonts/dinm.svg#dinm') format('svg');
}

p {
  font-family: dinm, Helvetica, Arial, serif;
  font-size: 4em;
  color: #ff00ff;
}

Color


Color can be applied using color names, hexadecimal codes, or RGBA/HSLA values, and can be applied to the foreground as well as background.

Color Types


Color Name

  • Plain text names
  • 140 available
  • DeepPink
  • DodgerBlue
  • GreenYellow

Hexadecimal

  • Uses alphanumeric code to specify RGB values.
  • 24-bit color
  • #FF1493
  • #1E90FF
  • #ADFF2F

RGBA/HSLA

  • Uses 8-bit numbers for color channels.
  • 32-bit color (can contain alpha channel).
  • rgb(255, 20, 147)
  • rgb(30, 144, 255)
  • rgb(173, 255, 47)
