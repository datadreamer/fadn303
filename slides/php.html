PHP Hypertext Preprocessor

  • Server-side scripting language.
  • Executes code on the server before returning HTML to the client.
  • Has variables, arrays, objects, conditionals, operators, and loops.
  • Used to connect to databases.
  • Mixes seamlessly with HTML.

Code must run on the server

Your PHP code will only run on a web server with PHP installed on it. You can run the code on the fadn303.net server or you can install a server on your laptop using MAMP.

Your PHP will not run by simply opening the file in your browser.

Separating PHP and HTML


All of your PHP code should go inside the php brackets.

<?php
  echo "This is how you print a line."
  echo "This is a line with a linebreak at the end.\n"
?>

Variables


Just like javascript, PHP can use variables to hold values.

<?php
  $a = 2;
  $b = 3;
  $c = $a + $b;
  echo "A plus B equals " . $c;
?>
A plus B equals 5

Operators


Operators perform math operations or comparisons.

<?php
  $a = 2;
  $b = 3;
  echo "A plus B equals " . ($a + $b) . "\n";
  echo "A minus B equals " . ($a - $b) . "\n";
  echo "A times B equals " . ($a * $b) . "\n";
  echo "A divided by B equals " . ($a / $b) . "\n";
?>
A plus B equals 5
A minus B equals -1
A times B equals 6
A divided by B equals 0.666666666667

Conditionals


Conditional statements use comparators to evaluate variables.

<?php
  $a = 7;
  if($a < 5){
    echo "A is less than five.\n";
  } else if($a > 5 && $a < 10){
    echo "A is greater than five but less than ten.\n";
  } else {
    echo "A is greater than ten.\n";
  }
?>
A is greater than five but less than ten.

Loops


Loops use variables and conditions to iterate over values.

<?php
  $a = 0;

Page Composition using PHP


You can use PHP to combine multiple HTML or PHP files together when a page is loaded.

<!doctype html>
<html>
  <head>
    <title>PHP Page Composition</title>
  </head>
  <body>
  <?php
    include("header.html");
    echo "My dynamic content goes here!<br/>";
    $a = 1;
    while($a <= 10){
      echo $a . "<br/>\n";
      $a++;
    }
    echo "And we out!<br/>\n";
    include("footer.html");
  ?>
  </body>
</html>

$_GET, $_POST and $_FILES

GET and POST are ways that variables can be passed from the client to the server upon accessing a web page. FILES is how files can be uploaded from the client to the server. This is basically how we transfer things through the browser.

$_GET

The $_GET[] associative array holds variables that are passed in through the URL.

<?php
  echo $_GET["test"];
?>
http://mw1.fadn303.net/demos/phpget/index.php?test=anything

$_POST

The $_POST[] associative array holds variables that are passed in using a POST transfer function. Variables are not passed via the URL, so they can be much larger and more private.

$_FILES

The $_FILES[] associative array holds references to temporary files that have been uploaded via a POST transfer.
