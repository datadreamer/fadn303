Block Elements

I'm inside a container!
Divs are used as containers to create divisions in your page content. They are block level elements.

I'm inside a container!
Divs inherit standard margins, padding, borders, and vertical and horizontal alignments used to contain content. All of these attributes can be changed with CSS.

Margins / Border / Padding


Elements can be positioned in different ways to respond to the page as well as other elements.

position: static;

  • Default position renders elements in order as they appear in the document flow.

position: relative;

  • The element is positioned relative to its normal position.
  • Adjusts based on left, right, top, and bottom CSS declarations.

position: absolute;

  • The element is positioned relative to its first positioned (not static) ancestor element.
  • Adjusts based on left, right, top, and bottom CSS declarations.
  • Like tacking an element to a specific position on the page.

position: fixed;

  • The element is positioned relative to the browser window.
  • Adjusts based on left, right, top, and bottom CSS declarations.
  • Like tacking an element to a specific position on the browser window.

Semantic Elements


Semantic elements help organize content for machine readability, making it easier to index in search engines.

  • <header>: Defines a header for a document or section.
  • <footer>: Defines a footer for a document or section.
  • <article>: Defines an article.
  • <section>: Defines a section in a document.
  • <nav>: Defines navigation links.


Check the complete HTML reference for more.

Inline Elements

Span Elements


Spans are used to position elements inline with other content such as text. Span elements can style individual text in a paragraph using a combination of CSS that italic and bold tags can not do alone. Unlike <div> elements, spans initialize as inline elements changing the way content wraps around them.
